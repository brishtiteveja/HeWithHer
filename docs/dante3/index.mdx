# Editor

## Installation

```shell
# with npm
npm install --save dante3
# with yarn
yarn add dante3
# with pnpm
pnpm install dante3
```

**Required**: 

```shell
# with npm
npm install --save prop-types
# with yarn
yarn add prop-types
# with pnpm
pnpm install prop-types
```

**Note**: Since Tiptap is based on Prosemirror, you will need to install the Prosemirror dependencies if you don't have them installed already to avoid version clashes.

**Note**: If you use NPM 7 or higher, peerDependencies are automatically installed.

```shell
# Installation with NPM 6 or lower
npm install --save prosemirror-commands prosemirror-keymap prosemirror-model prosemirror-schema-list prosemirror-state prosemirror-transform prosemirror-view prosemirror-history prosemirror-dropcursor prosemirror-gapcursor
# Installation with yarn
yarn add prosemirror-commands prosemirror-keymap prosemirror-model prosemirror-schema-list prosemirror-state prosemirror-transform prosemirror-view prosemirror-history prosemirror-dropcursor prosemirror-gapcursor
# Installation with pnpm
pnpm install prosemirror-commands prosemirror-keymap prosemirror-model prosemirror-schema-list prosemirror-state prosemirror-transform prosemirror-view prosemirror-history prosemirror-dropcursor prosemirror-gapcursor
```

## Basic usage

This is an empty editor. Will be initialized with default tooltips

```js live=true
  <Dante />
```


Is the same as

```js live=true
  <Dante content={null} />
```

## Body placeholder

```js live=true
  <Dante body_placeholder={"Do what you will" } 
  />
```


## Read Only

Read only will disable all interaction

```js live=true
  <Dante read_only={true} />
```


## onChange callback

onChange func will return the entire instance of the editor, so you can use all it's methods. 

### Save override example:

```js live=true
<Dante onChange={ (editor)=> console.log("editor content: ", editor.getHTML()) }/>
```
